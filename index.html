Master of Engineering

DISSERTATION: Development of a tool using Python for the Analysis of Static Aeroelasticity in Flexible Aircrafts.

ABSTRACT: This work presents the development of a computational tool, called Flying Circus using the Python programming language and open source libraries for static aeroelasticity analysis of aircraft in the conceptual development stage. The tool uses a vortex lattice method for aerodynamic calculations, a finite element method using Euler-Bernoulli beam elements for structural calculation, a proximity criterion for aerodynamic/structural coupling, and an iterative method for aeroelastic computation.
